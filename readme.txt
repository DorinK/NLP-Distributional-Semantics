  DistributionalSimilarities.py  

+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-

In the current working directory you should have:
1. DistributionalSimilarities.py
2. wikipedia.sample.trees.lemmatized
3. CoOccurrenceTypes.py
4. requirements.txt

First, please run:	pip install -r requirements.txt
And then:	 	python3 DistributionalSimilarities.py

The program does not take any parameters, it automatically computes 1st-order and 2nd-order similarities for each target word and for each co-occurrence type and finally prints these similarity tables.

	   Word2Vec.py

+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-

In the current working directory you should have:
1. Word2Vec.py
2. bow5.words
3. bow5.contexts
4. deps.words
5. deps.contexts
6. requirements.txt

First, please run:	pip install -r requirements.txt
And then:	 	python3 Word2Vec.py

The program does not take any parameters, it automatically computes 1st-order and 2nd-order similarities for each target word and for each vectors version and finally prints these similarity tables.
